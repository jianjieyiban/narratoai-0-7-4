# NarratoAI 配置说明

## 快速开始

1. **首次运行**：双击 `启动NarratoAI.bat`
2. **环境检测**：可先运行 `环境检测.bat` 检查环境
3. **访问应用**：启动后在浏览器打开 http://localhost:8501

## API密钥配置

在使用AI功能前，需要配置LLM API密钥：

### 1. 编辑配置文件
打开 `config.toml` 文件，配置以下部分：

```toml
# 视觉模型配置（用于视频理解）
vision_litellm_api_key = "你的API密钥"

# 文本模型配置（用于文案生成）  
text_litellm_api_key = "你的API密钥"
```

## 功能说明

### 核心功能
- ✅ AI自动生成影视解说文案
- ✅ 智能视频剪辑和片段选择
- ✅ 多引擎语音合成（TTS）
- ✅ 自动字幕生成
- ✅ Web图形化界面操作

### 支持的文件格式
- **视频**：MP4, AVI, MOV, MKV等
- **音频**：MP3, WAV, AAC等  
- **字幕**：SRT, ASS等

## 系统要求

- **操作系统**：Windows 10/11 或 macOS
- **内存**：8GB RAM（推荐16GB）
- **存储**：至少10GB可用空间
- **网络**：稳定的互联网连接（用于AI服务）

## 故障排除

### 常见问题

1. **Python未找到**：安装Python 3.12+并添加到PATH
2. **依赖安装失败**：检查网络连接，或手动安装
3. **端口占用**：修改`启动NarratoAI.bat`中的端口号
4. **API调用失败**：检查API密钥和网络连接
